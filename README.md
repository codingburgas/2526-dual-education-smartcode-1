                   🌿 SmartCode – Eco-City Explorer

 📌 Описание

SmartCode – Eco-City Explorer е интерактивен уебсайт, който симулира интелигентен и устойчив град чрез динамично съдържание, визуализации и геймификация.

Проектът предоставя богато потребителско изживяване чрез модерни UI техники като glassmorphism, динамични статистики, интерактивна карта и система за мисии.


 🎯 Основна идея

Целта на уебсайта е да позволи на потребителя да:

* изследва различни зони на „умен град“
* разбере как работят зелените технологии
* проследява своя прогрес
* взаимодейства с динамична среда

 ⚙️ Технологии

* **HTML5** – структура на страницата
* **CSS3** – custom properties, responsive дизайн, анимации
* **JavaScript (ES6)** – логика, DOM manipulation, интерактивност
* **localStorage** – запазване на прогрес и мисии


 ✨ Основни функционалности

 🗺️ Интерактивна карта

* 6 различни зони (соларен парк, гора, транспорт и др.)
* Клик върху зона → отваря modal с информация
* Възможност за „отключване“ на зони
* Визуален статус (explored / unexplored)


 📊 Динамични статистики

* Анимирани стойности (count-up ефект)
* Прогрес барове
* Симулация на реални данни


 🌫️ Air Quality Widget (уникална функционалност)

* Динамичен индекс на въздуха (AQI)
* Променя се спрямо действията на потребителя
* Визуална индикация за състоянието


 🎯 Mission System (геймификация)

* Мисии с условия (например: отключи 3 зони)
* Автоматично разпознаване на изпълнение
* Награди и визуална обратна връзка


 📔 Green Diary (прогрес система)

* Събиране на значки
* Филтриране по категории
* Процентен прогрес бар
* Запазване чрез localStorage


 🎬 Видео секция

* Вградени видеа (iframe)
* Динамична смяна на съдържание
* Плейлист система

 🌙 Eco Mode (тъмна тема)

* Превключване между светъл и тъмен режим
* Използване на CSS variables
* Запазване чрез localStorage

🎨 UI/UX характеристики

* Glassmorphism дизайн
* Responsive layout (mobile + desktop)
* Smooth scroll и transitions
* Hover ефекти и микро-анимации
* Modal прозорци
* Toast notifications

🧠 JavaScript концепции

* DOM manipulation
* Event listeners
* Работа с масиви и обекти
* Dynamic rendering (map, stats, diary)
* localStorage за състояние
* State management (zones, missions, filters)
